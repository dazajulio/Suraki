# Suraki - Supermarket Kiosk

Este es el proyecto del Kiosko Autoservicio de Suraki adaptado para Supermercados, integrado con Supabase y pasarela de pago Stripe.

## Primeros Pasos

Para correr el servidor de desarrollo local:

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver el resultado.
